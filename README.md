# Claude Code vs Codex — Head-to-Head Comparison

A small experiment comparing the output quality of **Claude Code** and **OpenAI Codex** when given the same minimal prompt: build a web-based todo list app.

## Goal

Evaluate how two AI coding assistants handle a simple, well-defined task from scratch — looking at code quality, structure, and usability of the result.

## Project Structure

```
todolist/
├── claudecodescodes/       # Output generated by Claude Code
│   ├── claudecode.html
│   └── calculator.html
├── codexcodes/             # Output generated by Codex
│   ├── codexcode.html
│   └── calculator.html
```

## The Prompt

> Create a web-based todo list.

Both assistants received the same prompt with no additional context or constraints.

## How to View

Open any `.html` file directly in a browser — no build step or server required.

## What We're Comparing

- Code quality and readability
- Feature completeness from a minimal prompt
- UI/UX out of the box
- Amount of boilerplate or unnecessary complexity
